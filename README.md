# PicoCTF Writeups

Welcome to my collection of **picoCTF** challenge writeups. This repository documents my learning journey as I develop practical cybersecurity skills through Capture The Flag (CTF) challenges.

Flags and challenge binaries are intentionally omitted to respect picoCTF's guidelines. The focus of this repository is on the methodology, tools, and concepts used to solve each challenge.

## About Me

I'm a Cybersecurity student at Purdue University with an interest in reverse engineering, digital forensics, binary exploitation, networking, and offensive security. This repository serves as both a study resource and a portfolio demonstrating my technical growth.



## Repository Structure

picoCTF/

├── Cryptography/
├── Forensics/
├── General-Skills/
├── Reverse-Engineering/
├── Web-Exploitation/

Each challenge folder typically contains:

* `README.md` or `writeup.md` – Challenge overview and solution methodology
* `solve.py` (when applicable) – Automation scripts
* `notes.md` – Additional observations and research
* Screenshots or diagrams (when helpful)

## Writeup Format

Each writeup includes:

* Challenge name
* Category
* Difficulty
* Objective
* Tools used
* Step-by-step methodology
* Commands executed
* Key concepts learned
* Lessons learned
